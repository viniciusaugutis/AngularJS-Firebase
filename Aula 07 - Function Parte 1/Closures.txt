Encerramento/Fechamento

var helloWorld = function(){
	var message = "oi";
	return function(){
		return message;
	};
};

console.log(helloWorld); //[function]
console.log(helloWorld()); //[function]
console.log(helloWorld()()); //Retorna o hello World normalmente

Vai mostrar o helloWorld na segunga corretamente, pois mantem a referencia do message que veio da função de cima, devido
as funcionalidades e recursos do closures. Assim consegue mostrar corretamente atraves da referencia externa do message.
